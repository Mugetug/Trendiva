# Sunum İçeriği

Bu projede Trendiva isimli mobil destekli e-ticaret sistemi geliştirilmiştir.

Projede aşağıdaki teknolojiler kullanılmıştır:

- React Native + Expo
- Node.js
- Express.js
- MongoDB Atlas
- Docker
- Jenkins CI/CD
- Redis Cache
- RabbitMQ

---

# Sunumda Gösterilecek Özellikler

## 1. Kullanıcı İşlemleri

- Kullanıcı kayıt olma
- Kullanıcı giriş yapma

## 2. Ürün İşlemleri

- Ürün listeleme
- Ürün detay görüntüleme
- Ürün görsellerini slider ile gösterme

## 3. Sepet İşlemleri

- Sepete ürün ekleme
- Sepeti listeleme
- Sepetten ürün kaldırma

## 4. Sipariş İşlemleri

- Sipariş oluşturma
- Sipariş sonrası RabbitMQ mesajı oluşturma

## 5. Redis Cache

- Ürün listeleme işlemlerinde Redis cache kullanımı
- Cache loglarının gösterilmesi

## 6. Docker

- Frontend container çalıştırılması
- Backend container çalıştırılması
- Redis container çalıştırılması
- RabbitMQ container çalıştırılması

## 7. CI/CD

- Jenkins pipeline çalıştırılması
- GitHub üzerinden otomatik build işlemleri
